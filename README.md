# 黑龙江省火灾格局分析项目

## 项目概述

本项目对黑龙江省2010-2019年期间的火灾格局进行了全面的时空维度分析，重点关注秸秆焚烧等火点的分布特征、季节性规律和空间聚集模式。通过多维度的数据分析和可视化，揭示了黑龙江省火灾发生的时间变化趋势、空间分布规律以及异常事件特征。

## 主要功能模块

### 1. 时间维度火灾格局分析

该模块主要分析火灾发生的时间规律和周期性特征，包括：

- **周期性统计分析**：按周、旬统计火点数量和平均FRP（火辐射功率）
- **高峰识别**：识别每年火灾高发的前3周，并进行统计分析
- **季节性特征分析**：绘制火点季节曲线，展示年度间的对比
- **异常事件检测**：识别火灾异常高峰周，为预警提供支持

### 2. 空间维度火灾格局分析

该模块主要分析火灾的空间分布特征和聚集模式，包括：

- **热点分析**：绘制多年平均火点热点图，识别火灾高发区域
- **密度分析**：生成年均火点密度分布图，展示空间聚集特征
- **地理信息处理**：利用县级行政边界数据，进行区域化分析

### 3. 秸秆焚烧特征专题分析

深入分析秸秆焚烧的时空分布特征，包括：

- **年度变化趋势**：分析2010-2019年秸秆焚烧的年度变化
- **季节性规律**：识别春季（3-5月）和秋季（9-11月）焚烧高峰
- **FRP（火辐射功率）分析**：统计不同时期的火强度特征
- **多维度可视化**：生成时间序列图、月度分布图、热力图等多种可视化结果

## 数据说明

### 数据源

项目使用的主要数据包括：

1. **MODIS火点数据**：2010-2019年黑龙江省MODIS卫星火点观测数据，包含火点位置、时间、火辐射功率等信息
2. **县级行政边界数据**：用于空间分析的基础地理信息数据

### 数据处理

原始数据经过清洗和预处理，生成了多个结构化的数据文件，包括：

- 按周、旬统计的火点数量和平均FRP数据表
- 高峰周和异常高峰周统计表
- 空间分析用的GeoDataFrame数据

## 主要发现

### 时间维度发现

1. **季节性明显**：黑龙江省火灾主要集中在春季（3-5月）和秋季（9-11月），这两个季节的火点数量占全年的大部分
2. **年度波动**：2010-2019年间，火灾发生数量存在明显的年际波动
3. **周度特征**：部分年份存在明显的火灾高峰周，需要重点关注

### 空间维度发现

1. **区域聚集性**：火灾分布呈现明显的空间聚集特征，某些区域成为火灾高发区
2. **热点分布**：多年平均火点热点图显示了火灾风险的空间分布格局
3. **密度差异**：不同地区的年均火点密度存在显著差异
